Sometimes is quite hard to figure what's eating up money in you Azure Subscriptions.
The Azure Cost Management tool is great, still, might also also be too great.

This repository is an attempt to showcase ways for you to see what's going on in you subscriptions via extracting cost information from your subscriptions in a more straightforward way.

# Prerequisites
- An Azure Subscription
- Azure CLI installed and configured or a copy of this repo and GitHub Actions enabled
- Familiarity with Azure, Biceps and/or GitHub Actions
